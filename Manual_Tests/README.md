# Manual Test Documentation

This directory contains documentation related to manual testing for the AMW Django ERP project.

## Contents:

-   **`CREDENTIALS.md`**: Securely stores credentials for various testing environments. (Note: This file should be handled with extreme care and never committed to public repositories.)
-   **`MANUAL_TEST_CASES.md`**: Outlines detailed manual test cases to be executed.
-   **`MANUAL_TEST_PLAN.md`**: Provides the overall plan and strategy for manual testing efforts.

Please refer to these documents for guidance on performing manual tests and understanding the testing scope.
